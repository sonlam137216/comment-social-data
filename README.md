# Comment social data code

## Installation

Use the package manager [yarn](https://classic.yarnpkg.com/lang/en/docs/install/#windows-stable/) to install packages.

```bash
yarn install
```

## Usage
To start the server to crawl data from Shopee
```bash
yarn server-axios
```
